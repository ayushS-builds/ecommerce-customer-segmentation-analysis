# ecommerce-customer-segmentation-analysis
End-to-end customer analytics project using SQL, Python, ML, and Power BI. Performed RFM analysis, churn detection, and K-Means clustering to segment customers. Built an interactive dashboard with key insights on revenue, risk segments, and customer behavior.
